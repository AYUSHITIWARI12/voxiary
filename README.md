# 🎭 Voxiary – Mood-Based Thought & Memory Journal

## 📌 About the Project
Voxiary is a full-stack journaling application that allows users to express their thoughts based on their mood and capture personal moments like a digital diary. Users can select emotions, write their feelings, and store memories securely.

---

## 🚀 Features
- 🎭 Mood-based journaling
- 📝 Thought narration
- 📸 Capture and store moments
- 🔐 PIN-based story locking
- 🔑 JWT & Google OAuth authentication
- 🔐 Encrypted content storage
- ✏️ Rich text editing

## ⚙️ How It Works
1. User logs in using JWT or Google OAuth
2. Selects mood and writes thoughts
3. Data is encrypted before storing
4. Stories can be locked with a PIN
5. Users can revisit and manage memories

---

## 🎯 Objective
To create a secure and emotional journaling platform where users can express thoughts and preserve memories.
